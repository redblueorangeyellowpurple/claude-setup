---
name: dashboard-qa
description: Validate dashboards, sheets, and multi-view UI changes
---

1. Identify all tabs/views affected by the change
2. Check layout, formatting, and alignment across all views
3. Validate data integrity:
   - metrics
   - calculations
   - aggregations
4. Verify filters, dropdowns, and controls work correctly
5. Compare against previous behavior to detect regressions
6. Check edge cases (empty data, partial data, date ranges)
7. Flag any inconsistencies before finalizing
