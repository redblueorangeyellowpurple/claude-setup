---
name: notion-format
description: Apply Notion formatting changes consistently across ALL tabs and sections in a single pass.
---

When formatting a Notion file or executive brief:

1. List every tab and section in the target file before touching anything
2. Apply all heading, color, bold, and rename changes to each tab in one pass — do not do them one at a time
3. After applying, grep for any remaining old patterns to verify nothing was missed
4. Confirm no tab or section was skipped before marking done
5. Commit and push only after all tabs are verified
