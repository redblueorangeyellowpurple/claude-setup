---
name: resume-note
description: Create a clean session handoff note with current state, changes, blockers, and exact next step.
---

Whenever a work session ends, produce:
1. What was attempted
2. What was completed
3. What changed
4. What failed or remains uncertain
5. Exact next step
6. A paste-ready resume prompt for the next session

Keep it short, specific, and actionable.
